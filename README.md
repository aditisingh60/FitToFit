# FoodToFit

A fitness and nutrition tracking application.

## Project Structure

- `frontend/`: React app built with Vite.
- `backend/`: Node.js + Express API.

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm start` (Make sure to add a start script)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
